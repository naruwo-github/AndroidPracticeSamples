#### Androidのアクティビティ
アクティビティは、Androidの「１つの画面」に対応するもの。アクティビティには、ビューの配置(ボタンや画像、テキストなど)やイベントの処理(キー操作や画面タッチ、ボタンクリックなど)などを書いていく。

### アクティビティとライフサイクル
アクティビティの状態は、大きく分けると４つ
・開始
・終了
・再開
・強制終了
